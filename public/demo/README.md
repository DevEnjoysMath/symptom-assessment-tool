## How to take good screenshots:

1. Use your browser's dev tools (F12) to set viewport to 1200x800
2. Hide the bookmarks bar for cleaner shots
3. Take screenshots at the same zoom level for consistency
4. For mobile view, use Chrome DevTools device mode (iPhone 14 Pro)

These images will appear in the README on GitHub!
